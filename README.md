# 👥 Employee Data Entry System

**Automated Excel Form for Employee Onboarding & Payroll Data Management**

## 📋 Project Overview

This project is a **custom automated Excel data entry form** designed to streamline the onboarding process for new employees. It captures personal details, department information, and payroll components in a user-friendly interface, with automatic data transfer to a structured database.

The tool ensures data accuracy, consistency, and security while saving significant time for HR teams.

## ✨ Key Features

- Clean and intuitive data entry form
- One-click **SUBMIT** automation (VBA-powered)
- Automatic Net Pay calculation
- Location selection (Site / Office)
- Protected sheet structure (only input fields are editable)
- All records stored in a clean **Database** sheet
- Macro-enabled (.xlsm) workbook

## 🛠️ How It Works

1. Open the Excel file
2. Fill in the employee information in the **Form** sheet
3. Click the **SUBMIT** button
4. Data is automatically appended to the **Database** sheet
5. Form clears for the next entry

## 🔒 Security Features

- Saved as **Macro-Enabled Workbook (.xlsm)**
- Sheet protection enabled
- Only input cells are unlocked
- Formulas and layout are locked to prevent accidental changes
